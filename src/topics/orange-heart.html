Orange Heart Project

Orange Heart Project 2024 explores themes of addressing and belonging. This collection of images viewed through a heart-shaped stencil was created from May - August 2024.

You are viewing this content live on the Interplanetary File System (IPFS). The current website is more stable and resilient than if it was only being stored in a single location on the web.

The source content was developed in DITA XML and travelled through a typical enterprise content development lifecycle using the DITA Open Toolkit, Github and Fleek.

The content that you are viewing is stored on the Interplanetary File System, IPFS , which ensures that the content is decentralized, replicated, and verifies its authenticity and provenance.

I first started stenciling hearts in my neighbourhood as part of a healing journey. Since then, they have been reused and repurposed on a variety of mediums beyond concrete.

This project repurposes a heart-shaped stencil originally created as a response to gun violence in San Francisco's Mission District in 2017.
