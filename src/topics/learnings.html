Learnings

Answers to a lot of questions that I had during this project are gathered here.

My project goals were to experiment and try out different Dweb-friendly tools and technologies.

Source content is written in OASIS Open DITA XML transformed to HTML5 using the DITA Open Toolkit 4.2.3 with Github Actions. HTML5 mods performed by Python, rendered with Responsive Bulma CSS and deployed to the InterPlanetary File System (IPFS) through Fleek.

What processes need to change if you want to publish content to IFPS?
None. Using Fleek as my platform provider, meant that I could simply auto-deploy from Git. When experimenting with the DWeb YVR chat app, I used the Fleek CLI. Having Fleek handle the entire IPFS layer is a happy path.
How much do I need to think about security and encryption? Or is it auto-magical?
It is auto-magical https when deploying from Fleek.
How do I deploy a dweb site?
Select a platform provider or you can publish directly onto IPFS yourself.
How do I make the references to graphics, should I be using CIDs rather than URLs?
Human readable filenames and human-readable URLS are fine.
