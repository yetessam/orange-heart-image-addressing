Deployment

Platforms and approaches to deploying HTML to the decentralized web. Multiple platforms and services can publish, pin or host the content using IPFS.

The following information is current as-of January 2025 and was generated by GitHub Copilot.

  1. Fleek: Provides seamless integration with IPFS and offers tools for deploying, hosting, and managing websites on the decentralized web.
  2. Pinata: Offers pinning services for IPFS, allowing you to store and manage files on IPFS with additional tools for developers.
  3. Infura: Provides IPFS and Ethereum APIs, enabling easy interaction with IPFS for hosting and retrieving content.
  4. Temporal: Offers IPFS pinning and hosting services with additional features like versioning and file management.
  5. Cloudflare: Offers an IPFS gateway, enabling access and interaction with IPFS through Cloudflare's infrastructure.
  6. Web3.Storage: A service provided by Protocol Labs for storing data on IPFS with easy-to-use APIs and integration.
