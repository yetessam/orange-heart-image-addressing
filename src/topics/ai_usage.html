    Discover how Artificial Intelligence (AI) was used in this project.

    AI was an excellent learning partner and researcher on the Dweb stack. Copilots were used for programming tasks such as prototyping, refactoring code, rewriting existing functions, debugging and scanning through complex messages and logs. For content development, I wrote the content myself. Personally, I prefer my short, fact-based minimalism to the rambling, trippy Generative AI content.

    Assistance Type Example (s) Technology
    Learning and research partner
    • Provided concise summaries to help compare different dweb technologies
    • Up-to-date knowledge on latest dweb technology
    • Interactive learning partner when prompted with learning objectives
    		Open AI GPT-4o, GPT-3.5,
    Programming
    • Troubleshooting error messages
    • Scanning build logs
    • Used for prototyping Python and helped set up Github actions
    		Open AI GPT-4o,GPT-4, GPT-3.5, GitHub Copilot, Anthropic Claude 3.5 Sonnet, VS Code Copilot, Microsoft Copilot
    Content development
    • Formatting code and XML nesting
    • Adding XML tagging with auto-complete in VS Code
    • Spelling and grammar checking
    		Open AI GPT-4o, GitHub Copilot, VS Code Copilot
